## 初始化

### 字符串初始化

```python
from pyquery import PyQuery as pq

html = """
<div>
<ul>
<li class="item-0">first item</li>
<li class="item-1"><a href="link2.html">second item</a></li>
<li class="item-0 active"><a href="link3.html"><span class="bold">third item</span></a></li>
<li class="item-1 active"><a href="link4.html">fourth item</a></li>
<li class="item-0"><a href="link5.html">fifth item</a></li>
</ul>
</div>
"""

doc = pq(html)
print(doc('li'))
```
运行结果：
```
<li class="item-0">first item</li>
<li class="item-1"><a href="link2.html">second item</a></li>
<li class="item-0 active"><a href="link3.html"><span class="bold">third item</span></a></li>
<li class="item-1 active"><a href="link4.html">fourth item</a></li>
<li class="item-0"><a href="link5.html">fifth item</a></li>
```

### URL初始化

```python
from pyquery import PyQuery as pq

doc = pq(url='http://balbozheng.com')
print(doc('title'))
```
运行结果：
```
<title>ZhengXuJi</title>
```
也可以写成
```python
from pyquery import PyQuery as pq
import requests

doc = pq(requests.get('http://balbozheng.com').text)
print(doc('title'))
```

### 文件初始化

```python
from pyquery import PyQuery as pq

doc = pq(filename='demo.html')
print(doc('li'))
```

## 遍历

## 获取信息
### 获取属性

提取某个节点后，我们可以调用 attr() 方法来获取属性
```python
from pyquery import PyQuery as pq

html = """
<div class="wrap">
<div id="container">
<ul class="list">
<li class="item-0">first item</li>
<li class="item-1"><a href="link2.html">second item</a></li>
<li class="item-0 active"><a href="link3.html"><span class="bold">third item</span></a></li>
<li class="item-1 active"><a href="link4.html">fourth item</a></li>
<li class="item-0"><a href="link5.html">fifth item</a></li>
</ul>
</div>
</div>
"""

doc = pq(html)
a = doc('.item-0.active a')
print(a, type(a))
print(a.attr('href'))
```
运行结果：
```
<a href="link3.html"><span class="bold">third item</span></a> <class 'pyquery.pyquery.PyQuery'>
link3.html
```
如果要获取多个节点属性，我们就要使用遍历
```python
for item in a.items():
    print(item.attr('href'))
```
运行结果：
```
link2.html
link3.html
link4.html
link5.html
```

### 获取文本
获取其内部文本
```python
a = doc('.item-0.active a')
print(a)
print(a.text())
```
运行结果：
```
<a href="link3.html"><span class="bold">third item</span></a>
third item
```
如果想获取这个节点内部的 HTML 文本，就要用 html() 方法
```python
a = doc('.item-0.active a')
print(a)
print(a.html())
```
运行结果：
```
<a href="link3.html"><span class="bold">third item</span></a>
<span class="bold">third item</span>
```
如果我们选中的结果是多个节点，想要获取每个节点的内部 HTML 文本，则需要遍历每个节点。而 text() 方法你休安排遍历就可以获取
```python
li = doc('li')
print(li.html())
print(li.text())
print(type(li.text()))
```
运行结果：
```
first item
first item second item third item fourth item fifth item
<class 'str'>
```

## 节点操作
### addClass 和 removeClass

```python
li = doc('.item-0.active')
print(li)
li.remove_class('active')
print(li)
li.add_class('active')
print(li)
```
运行结果：
```
<li class="item-0 active"><a href="link3.html"><span class="bold">third item</span></a></li>
<li class="item-0"><a href="link3.html"><span class="bold">third item</span></a></li>
<li class="item-0 active"><a href="link3.html"><span class="bold">third item</span></a></li>
```
addClass() 和 removeClass() 这些方法可以动态改变节点的 Class 属性


### attr、text 和 html

```python
from pyquery import PyQuery as pq

html = """
<ul class="list">
<li class="item-0 active"><a href="link3.html"><span class="bold">third item</span></a></li>
</ul>
"""

doc = pq(html)
li = doc('.item-0.active')
print(li)
li.attr('name', 'link')
print(li)
li.text('changed item')
print(li)
li.html('<span>changed item</span>')
print(li)
```
运行结果：
```
<li class="item-0 active"><a href="link3.html"><span class="bold">third item</span></a></li>
<li class="item-0 active" name="link"><a href="link3.html"><span class="bold">third item</span></a></li>
<li class="item-0 active" name="link">changed item</li>
<li class="item-0 active" name="link"><span>changed item</span></li>
```

### remove()

```python
from pyquery import PyQuery as pq

html = """
<div class="wrap">
    Hello World
<p>This is a paragraph.</p>
</div>
"""

doc = pq(html)
wrap = doc('.wrap')
print(wrap.text())
wrap.find('p').remove()
print(wrap.text())
```
运行结果：
```
Hello World
This is a paragraph.
Hello World
```

## 伪类选择器
CSS 选择器之所以强大，其中之一是因为他支持多种多样的伪类选择器

```python
from pyquery import PyQuery as pq

html = """
<div class="wrap">
<div id="container">
<ul class="list">
<li class="item-0">first item</li>
<li class="item-1"><a href="link2.html">second item</a></li>
<li class="item-0 active"><a href="link3.html"><span class="bold">third item</span></a></li>
<li class="item-1 active"><a href="link4.html">fourth item</a></li>
<li class="item-0"><a href="link5.html">fifth item</a></li>
</ul>
</div>
</div>
"""

doc = pq(html)
li = doc('li:first-child')
print(li)
li = doc('li:last-child')
print(li)
li = doc('li:nth-child(2)')
print(li)
li = doc('li:gt(2)')
print(li)
li = doc('li:nth-child(2n)')
print(li)
li = doc('li:contains(second)')
print(li)
```
运行结果：
```
<li class="item-0">first item</li>
<li class="item-0"><a href="link5.html">fifth item</a></li>
<li class="item-1"><a href="link2.html">second item</a></li>
<li class="item-1 active"><a href="link4.html">fourth item</a></li>
<li class="item-0"><a href="link5.html">fifth item</a></li>
<li class="item-1"><a href="link2.html">second item</a></li>
<li class="item-1 active"><a href="link4.html">fourth item</a></li>
<li class="item-1"><a href="link2.html">second item</a></li>
```
