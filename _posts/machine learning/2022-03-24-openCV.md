## openCV基础操作

### 图像的基础操作

1. 加载图像

   ```python
   # flags= {1、 0、-1}
   # 1 彩色图像
   # 0 灰度图像
   # -1 aplha通道的加载图像
   img = cv2.imread('val2017/val2017/000000001000.jpg', flags=-1)
   ```

2. 显示图像

   ```python
   # opencv
   cv2.imshow('image', img)
   cv2.waitKey(0)
   
   # matplotlib
   plt.imshow(img[:, :, ::-1])
   ```

3. 保存图像

   ```python
   cv2.imwrite('image.png', img)
   ```

#### 绘制几何图形

1. 绘制直线

   ```python
   # pt1,pt2 直线起点和终点
   # thickness 线条宽度
   # lineType 线型
   cv2.line(img=, pt1=, pt2=)
   ```

2. 绘制圆形

   ```python
   # center 圆心
   # radius 半径
   cv2.circle(img=, center=, radius=)
   ```

3. 绘制矩形

   ```python
   # pt1,pt2 直线起点和终点
   # thickness 线条宽度
   # lineType 线型
   cv2.rectangle(img=, pt1=, pt2=)
   ```

4. 添加文字

   ```python
   # text 文本
   # org 位置
   # fontFace 字体
   cv2.putText(img=, text=, org=, fontFace=)
   ```

#### 获取并修改图形像素点

```python
# 获得某个像素点的值
px = img[100, 100]
# 仅获取蓝色通道的值
px = img[100, 100, 0]
# 修改某个位置的像素值
img[100, 100] = [200, 200]
```

#### 图像通道的拆分合并

```python
# 通道拆分
b, g, r = cv2.split(img)
# 通道合并
img = cv2.merge((b, g, r))
```

#### 色彩空间的改变

```python
# flag {cv2.COLOR_BGR2GRAY, cv2.COLOR_BGR2HSY}
cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)
```
